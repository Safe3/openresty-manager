安装指南

简单几步，快速部署OpenResty Manager

主机版安装

sudo bash -c "$(curl -fsSL https://om.uusec.com/installer_cn.sh)"

访问地址：http://ip:34567

默认用户名：admin

默认密码：Passw0rd!

Docker版安装

curl https://om.uusec.com/docker_cn.tgz -o om.tgz && tar -zxf om.tgz && sudo bash ./om/om.sh

访问地址：http://ip:34567

默认用户名：admin

默认密码：Passw0rd!
